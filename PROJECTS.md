## 🔥 Core Projects

### 🚗 FiveM Car Internet Radio System  
🔗 https://github.com/NekoSuneProjects/car-internet-radio-fivem

A full-stack radio system for FiveM servers with real backend infrastructure.

**Stack:**
- Node.js backend
- PostgreSQL database
- React + Tailwind admin panel
- FiveM (Lua + xsound integration)

**Features:**
- Dynamic radio stations (admin-managed)
- JWT authentication + security systems
- In-game UI with live streaming audio
- Admin dashboard for full control

👉 This is one of my most complete full-stack systems. :contentReference[oaicite:0]{index=0}

### 🤖 VRChat Group Logger / Moderation Bot  
🔗 https://github.com/NekoSuneProjects/VRChat-Group-Logger

Advanced Discord bot for VRChat group moderation and logging.

**Stack:**
- Node.js
- Discord API (slash commands)
- MySQL + SQLite hybrid storage

**Features:**
- Ban / unban / moderation workflows
- Avatar, user, and group blacklist system
- Staff role + permission management
- VRChat API integration
- Helpdesk + ticket system support
